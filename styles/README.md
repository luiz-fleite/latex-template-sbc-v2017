# Diretório de Estilos (`styles/`)

Este diretório contém os pacotes e definições de formatação personalizados do documento.

- **`sbc-template.sty`**: O arquivo de estilo principal (`.sty`) da Sociedade Brasileira de Computação (SBC). Ele define as margens, fontes, cabeçalhos e comandos específicos (como `\address`, `\resumo`) utilizados no `main.tex`.
