# Diretório de Bibliografia (`bib/`)

Este diretório concentra os arquivos relacionados às referências bibliográficas do projeto.

- **`sbc-template.bib`**: O "banco de dados" das referências. Aqui você adiciona os livros, artigos e sites que citará no texto (formato BibTeX).
- **`sbc.bst`**: O arquivo de estilo bibliográfico (`.bst` - BibTeX Style). Ele define as regras de formatação visual das citações (negrito, itálico, ordem ABNT/SBC) na lista final de referências.
