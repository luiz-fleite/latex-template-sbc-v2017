# Diretório de Imagens (`images/`)

Este diretório armazena todos os recursos visuais (figuras, fotos, diagramas, gráficos) utilizados no artigo.

- **Uso no LaTeX:** As imagens devem ser importadas usando `\includegraphics{nome-do-arquivo}`. Não é necessário especificar o caminho completo, pois o `main.tex` já configura `\graphicspath{{images/}}`.
- **Formatos Aceitos:** JPG, PNG, PDF, EPS.
